# ETAPA 02 — AGENTS e estrutura local

## Objetivo
Registrar o alinhamento do repositorio local com a estrutura-base documentada e adicionar instrucoes operacionais persistentes para o agente.

## Arquivos e pastas criados
- `AGENTS.md`
- `app/`
- `data/catalogs/`
- `data/galleries/`
- `data/imports/`
- `data/logs/`
- `scripts/`
- `config/`
- `backups/`
- arquivos `.gitkeep` nas pastas vazias para permitir rastreamento pelo Git

## Por que `AGENTS.md` foi adicionado
O arquivo foi criado para registrar regras operacionais persistentes do projeto, incluindo objetivo do sistema, limites de atuacao do agente e a exigencia de trabalhar em etapas pequenas com revisao humana antes de passos maiores.

## Confirmacao de escopo
Nenhum arquivo fora deste repositorio foi alterado. Nenhuma dependencia foi instalada. Nenhuma mudanca de infraestrutura foi realizada.
