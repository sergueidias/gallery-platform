# Project Map

## Estrutura

app/ → código da aplicação  
data/ → dados do sistema  

### data/
catalogs/ → separação por domínio  
galleries/ → galerias publicadas  
imports/ → export do Lightroom  
logs/ → logs do sistema  

scripts/ → automações  
config/ → configurações  
backups/ → backups  

## Fluxo

Lightroom → imports → processamento → galleries → acesso via web
